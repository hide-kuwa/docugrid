# 画面設計ガイド（非エンジニア向け）

コードを書かずに、画面の見え方を3段階でカスタマイズできます。

## 3つの層（上から順に上書き）

```
① 全体デフォルト（プラットフォーム）
      ↓ 上書き
② 事務所（会社）ごと
      ↓ 上書き
③ 自分専用
      ↓
   実際に表示される画面
```

| 層 | 誰が編集 | どこで編集 |
|----|----------|------------|
| 全体デフォルト | 運営（platform_admin） | 設定 → 画面設計 →「全体デフォルト」 |
| 事務所ごと | 事務所管理者 | 設定 → 画面設計 →「事務所（会社）ごと」 |
| 自分専用 | ログイン中の本人 | ワークスペース内「自分専用の画面設計」 |

## 編集できる項目（枠）

- **ページタイトル** … 画面上部の大きな見出し
- **歓迎メッセージ** … タイトル下の説明文
- **アクセント色** … 画面上部の色ライン
- **ウィジェット** … 今後、ボタンや一覧の枠として増やします（全体デフォルトは JSON ファイルでも編集可）

## 設定手順（事務所管理者）

1. ログイン → **設定**（歯車）
2. 左メニュー **画面設計**
3. **対象ペルソナ** を選ぶ（例: クライアント・担当経理）
4. **事務所（会社）ごと** タブを開く
5. タイトル・メッセージを入力 → **この層を保存**
6. 右の **合成プレビュー** で最終的な見え方を確認

## 設定手順（一般ユーザー）

1. ログイン（クライアント経理などワークスペース画面へ）
2. **自分専用の画面設計** を開く
3. タイトル・メッセージを変更 → **保存**

自分専用の設定は、事務所の設定より優先されます。

## ファイルの場所（上級者・バックアップ用）

| 層 | ファイル |
|----|----------|
| 全体 | `backend/storage/platform/screen_design.json` |
| 事務所 | `backend/storage/firms/{事務所ID}/screen_design.json` |
| 自分 | `backend/storage/firms/{事務所ID}/members/{メンバーID}/screen_design.json` |

初回起動時は `screen_design.json.example` がコピーされます。
